# About this project

This is an authentication app, which provides the ability to register an account with email and password or log in with an existing account, including a Google account. All created data is stored in Firebase. The authenticated state is stored using Redux Toolkit so that a user stays logged in on every page.

### Stack:

1. React.js;
2. React Router v.6;
3. React Redux;
4. Redux Toolkit;
5. Firebase SDK.
